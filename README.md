# 🧠 FinAI - Personal Finance AI Assistant

FinAI is a comprehensive, AI-powered personal finance management application designed to help users take control of their financial lives. Built with modern web technologies and featuring an intuitive, eye-catching glassmorphism interface, FinAI combines powerful analytics with intelligent recommendations to deliver a complete financial wellness solution.

![FinAI Dashboard](screenshot.png)

## ✨ Key Features

### 🤖 AI Financial Advisor
- **Interactive Chatbot**: Real-time conversational AI providing personalized budgeting, investment, and savings advice
- **Smart Insights**: Automated analysis of spending patterns with actionable recommendations
- **Contextual Responses**: AI understands queries about savings, investments, budgeting, and goals
- **Quick Actions**: One-click suggestions for common financial questions

### 📊 Real-time Dashboard
- **Beautiful Glassmorphism UI**: Modern frosted glass design with gradient accents
- **Live Financial Metrics**: Animated counters for balance, income, expenses, and savings
- **Visual Analytics**: Interactive charts using Chart.js for trend analysis
- **Dark Mode**: Elegant dark theme optimized for extended use

### 💳 Expense & Income Tracking
- **Transaction Management**: Add, categorize, and monitor all financial transactions
- **Smart Categorization**: Auto-categorization with 10+ predefined categories
- **Filtering System**: Filter by income, expense, or view all transactions
- **Status Tracking**: Monitor transaction completion status

### 🎯 Budget Management
- **Monthly Budgets**: Set spending limits across multiple categories
- **Visual Progress Bars**: Real-time budget utilization with color-coded alerts
- **Overspending Warnings**: Automatic alerts when exceeding budget limits
- **Remaining Calculations**: Track exactly how much is left per category

### 🏦 Savings Goals
- **Goal Creation**: Create unlimited savings goals with custom icons and targets
- **Progress Tracking**: Visual progress indicators with percentage completion
- **Fund Addition**: Add funds to goals directly from the dashboard
- **Deadline Tracking**: Countdown timers showing days remaining until target date

### 📈 Investment Portfolio
- **Multi-Asset Support**: Track stocks, cryptocurrencies, and ETFs
- **Performance Metrics**: Real-time price changes and percentage gains/losses
- **Portfolio Overview**: Total value calculations with share counts
- **Visual Indicators**: Color-coded performance badges

### 🔔 Bill Reminders
- **Upcoming Bills**: Never miss a payment with due date tracking
- **Urgency Indicators**: Visual alerts for bills due within 3 days
- **Payment Status**: Mark bills as paid with one click
- **Amount Tracking**: Monitor recurring payment amounts

### 📱 User Experience
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Fade-ins, slide-ups, pulses, and hover effects
- **Custom Scrollbars**: Styled scrolling for better aesthetics
- **Notification System**: Toast notifications for user actions
- **Local Storage**: All data persists across browser sessions

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server or build tools required!

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/finai-personal-finance-ai.git
   ```

2. **Navigate to project directory**
   ```bash
   cd finai-personal-finance-ai
   ```

3. **Open in browser**
   ```bash
   # Simply open index.html in your preferred browser
   open index.html
   ```
   
   Or serve with a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```

4. **Start managing your finances!** 💰

## 🎯 Usage Guide

### Adding Transactions
1. Click **"Add Transaction"** button
2. Select type (Income/Expense)
3. Enter amount, description, category, and date
4. Save to update dashboard automatically

### Chatting with AI Advisor
1. Click the **floating robot button** or **"Ask AI"**
2. Type your financial question or use quick suggestions
3. Receive personalized advice based on your data

### Creating Savings Goals
1. Navigate to **Savings Goals** section
2. Click **"New Goal"**
3. Set target amount, current savings, and deadline
4. Track progress and add funds anytime

### Managing Budgets
- View real-time budget utilization
- Identify overspending categories (marked in red)
- Adjust spending habits based on AI recommendations
